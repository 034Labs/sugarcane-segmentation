# 🎋 Sugarcane Segmentation (Segmentação de Toletes de Cana-de-açúcar)

Este projeto tem como objetivo treinar e utilizar um modelo de segmentação baseado no **RF-DETR** para detectar e segmentar com precisão toletes de cana-de-açúcar.

## 🎯 Objetivo

O foco principal é aplicar técnicas avançadas de visão computacional (segmentação de instâncias) para identificar e delimitar os toletes de cana.

## 🛠️ Tecnologias Utilizadas

- Python
- PyTorch
- RF-DETR (Robust Feature Detection Transformer)
